## 🎯 Project Overview

**Student Portal** is a comprehensive web-based application designed to streamline student management and online testing processes. This project serves as a complete solution for educational institutions to manage student registrations, conduct online tests, and generate instant result sheets.

### 🎨 Problem Statement
Traditional manual systems for student management and test evaluation are time-consuming, error-prone, and lack efficiency. This project addresses these challenges by providing a digital platform that automates:

- Student registration and profile management
- Unique identification (auto-generated roll numbers)
- Online test administration
- Instant result calculation and grading
- Detailed performance analysis

### 🌟 Objectives
1. To create a user-friendly interface for student registration
2. To implement secure and unique roll number generation
3. To develop an efficient online testing system with timer
4. To provide instant result calculation with detailed analysis
5. To enable marksheet download for future reference

## ✨ Key Features

### 📝 Student Management
| Feature | Description |
|---------|-------------|
| **Multi-Step Registration** | Two-step process for complete information collection |
| **Document Upload** | Support for photo and ID proof upload |
| **Auto Roll Number** | Unique identification format: `BCA/2026/XXXX` |
| **Profile Management** | Complete student information dashboard |

### 📊 Online Test System
| Feature | Description |
|---------|-------------|
| **MCQ Format** | 10 multiple-choice questions |
| **Timer Integration** | 30-minute countdown with visual alerts |
| **Progress Tracking** | Real-time progress bar |
| **Auto-Save Answers** | Automatic answer preservation |
| **Instant Evaluation** | Immediate score calculation |

### 📈 Result Analysis
| Feature | Description |
|---------|-------------|
| **Score Calculation** | Percentage-based scoring system |
| **Grade Allocation** | A+ to D grade based on performance |
| **Answer Review** | Detailed comparison with correct answers |
| **Download Option** | HTML format marksheet download |

### 🎨 User Experience
| Feature | Description |

| **Dark Theme** | Modern, eye-friendly interface |
| **Smooth Animations** | Professional transitions and effects |
| **Responsive Design** | Perfect on desktop, tablet, and mobile |
| **Navigation Tabs** | Easy switching between sections |
